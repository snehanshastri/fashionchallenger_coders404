Greetings!

This is the official github repository of The Fashion Challenger System built on account of the Myntra Hackerramp - We for She. The system includes:
  A landing page
  A login page
  A sign up page
  A fashion challenges viewer page
  An uploader page
  A submissions gallery
  A leaderboard 
  A rewards page
